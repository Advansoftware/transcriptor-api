Documentação da API

Use este comando para transcrever um áudio via terminal:

curl -X POST https://transcriptor.respondia.pro/transcribe \
  -H "X-API-Key: SUA_CHAVE_AQUI" \
  -F "file=@/caminho/do/seu/audio.mp3" \
  -F "model=base"

Exemplo de integração usando Python:

import requests

url = "https://transcriptor.respondia.pro/transcribe"
headers = {"X-API-Key": "SUA_CHAVE_AQUI"}
data = {"model": "small"} # Escolha entre tiny, base, small, medium, large
files = {"file": open("audio.mp3", "rb")}

response = requests.post(url, headers=headers, data=data, files=files)
print(response.json()["text"])

No painel da Evolution API, adicione estes campos no Webhook:

URL: https://transcriptor.respondia.pro/transcribe

Cabeçalho: X-API-Key: SUA_CHAVE_AQUI

Parâmetros Form-Data: model=base

Recomendamos o modelo small para melhor captura de áudios de voz do WhatsApp.
